**[Apache RocketMQ](https://rocketmq.apache.org) 是一个延迟低、高性能、灵活性高，高可用万亿级容量分布式消息流平台**

本版本为RocketMq4.3.2
,它提供多种特色功能:

* 发布/订阅消息模型
* 定时、延迟消息
* 通过时间或偏移量 消息回溯
* 日志收集
* 大数据整合
* 在同一队列中支持可靠的FIFO和严格有序的消息传递
* 高校的拉取、推送消费模型
* 支持多种消费协议，如JMS 、OpenMessaging
* 支持分布式可扩展的部署架构
* 轻量、快捷的批量消息传输
* 多种消息过滤器，如tag
* 丰富的后台管理，如配置、指标、监控等
